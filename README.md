## 서비스 소개
#### 1인가구 및 지역 소상공인을 위한 공동구매 서비스 ${\textsf{\color{yellowgreen}SURROUND}}$  
<p></p>

&ensp; 📍 `위치 기반` 동네 설정으로 이웃과 생필품 소비 비용 절감
&ensp; 👤 `성별 필터` 를 통한 안전한 거래 진행  
&ensp; 🌳 서비스 내 `커뮤니티` 를 통한 지역 교류 형성
